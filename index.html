HTML, CSS, and JavaScript. Period.

Use any tool you want. With Lorem you don't need to learn anything new. No special syntax, no documentation, no fitting into a box. Just HTML (with any CSS or JavaScript you want). 

A better CMS for your website

Lorem analyzes your HTML to identify what is editable and automatically builds a visual editing interface for your site. This enables the rest of your team to edit everything they want, manage the site, and publish changes without shoulder-tapping developers. We’ve made content editing as simple as possible.

Hosted by perfectionists.

With Lorem you don't need to think about hosting (or speed or uptime or security). We meticulously follow best practices to ensure that every one of our sites is running on the best hosting infrastructure possible. Statically served from a global CDN with SSL, your site will be as fast and secure as websites can get.

git everything.

Lorem maps every change made in the visual editor back to version control to keep everything in sync. You can git pull to get the latest changes, git push to deploy a new version, git branch to create a draft and show off your progress, or git revert to a previous version at any time.
